# Particle Container Simulation

This project is a simple 2D simulation of particles moving inside a bounded container. It was made only for studying purposes

## Description

* Particles move with a constant velocity inside a rectangular container
* When a particle reaches a wall, it bounces back by reversing its direction
* The simulation is rendered in the console using a grid representation (only visual)

## Current Features

* Single and multiple particle support
* Wall collision handling
* Continuous movement loop

## Next Steps

This project is still in development. I plan to update it later to include:

* Particle–particle collision detection
* More accurate physics behavior
* Performance improvements (e.g., spatial partitioning)
* Better visualization
* Random particle position to avoid hardcoding
